AV System Drawing Package
Divisible Conference Center (Sample Project)

This 58-sheet drawing set showcases a full audiovisual system design for a theoretical government-grade divisible conference center. The project includes three multipurpose rooms, a central AV control room, and supporting spaces such as a recording studio, lobby displays, and secure conferencing suites.

  • Scalable AV infrastructure design across multiple room types
  • Integration of signal flow diagrams, device schedules, cable schedules, and rack elevations
  • Secure routing concepts supporting multi-classification (SIPR/NIPR) workflows
  • Standards-compliant drafting conventions and custom title blocks
  • Realistic device selection and labeling consistent with enterprise deployments

This drawing set departs from traditional left-to-right, signal-type-separated AV documentation in favor of a centralized, field-optimized layout. Devices are grouped by physical location and logical relationship, with signal paths laid out clearly and linearly to minimize flags, wire tags, and page jumps. This approach significantly reduces installation confusion, speeds up commissioning, and improves long-term maintainability ‚Äî especially in environments where the original integrator is no longer involved. While not yet common in the AV industry, this documentation style reflects a practical, technician-centered design philosophy.

This design is conceptual and does not represent a real-world facility. It is presented here for demonstration purposes only.
